# ⚔ Dungeon Crawler

A minimalist roguelike in your terminal.

## Install & run

```bash
npm install
npm start
```

## Controls

| Action / Action        | Keys                        |
|------------------------|-----------------------------|
| Move / Déplacer        | `↑↓←→` · `WASD` · `ZQSD` · `hjkl` |
| Use potion / Potion    | `P`                         |
| Quit / Quitter         | `Escape`                    |

> WASD for QWERTY keyboards, ZQSD for AZERTY.

## Stack

- [blessed](https://github.com/chjj/blessed) — terminal UI
- [rot-js](https://github.com/ondras/rot.js) — map generation, FOV, pathfinding
